Features

Everything Currencies can do, explained in one place.

Currency Conversion

🌐

Multiple Providers

Choose from 6 active exchange-rate providers. Switch at any time in Settings — the next refresh uses the new source.

⭐

Starred Currencies

Mark currencies as favourites to keep your most-used ones at the top. Tap the star icon next to any currency to toggle.

🔄

Swap & Mirror

Instantly swap the base and target currencies. The conversion result updates in real time without re-entering values.

📋

Copy / Paste

Long-press the result to copy it to the clipboard, or paste a value directly into the input field.

Calculator

🧮

Full Arithmetic

Addition, subtraction, multiplication, and division — evaluated before conversion using the mXparser engine.

⌨️

Extended Keypad

Optional expanded keypad mode adds operator keys directly on the main screen. Enable in Settings → Extended keypad.

🖥️

Hardware Keyboard

Physical keyboard input is fully supported — great for tablets and foldable devices with an attached keyboard.

Fee Calculator

💳

Foreign Exchange Fee

Add a configurable fee percentage (e.g., 1.5 %) to every conversion. The result shows both the raw and fee-inclusive amounts simultaneously.

🔧

Custom Percentage

Set any fee from 0 % to any value in Settings. Useful for credit-card foreign-transaction fees or bureau-de-change margins.

Historical Rates & Charts

📅

Past Dates

Select any historical date using the built-in date picker. Data availability depends on the provider — Frankfurter.app goes back to 2010-01-04.

📈

1-Year Chart

Tap the chart icon on the converter to see a full year of daily rate history for the selected currency pair.

🔍

Interactive Timeline

Scrub along the chart to see exact rates on any day. High and low points are highlighted for quick reference.

UI & Customisation

🎨

Material 3 Design

Built with the latest Material Design system components — adaptive colours, proper elevation, and dynamic theming.

🌙

Dark & Pure Black

Three theme options: Light, Dark, and Pure Black (OLED-friendly). Follows system setting or can be locked manually.

📱

Foldable Support

Detects fold state via WindowInfoTracker and switches to a multi-pane layout when unfolded, making full use of the larger screen.

🌏

20+ Languages

Translated by the community via Weblate. RTL languages are fully supported.

👆

Predictive Back Gesture

On Android 13+ the system's animated back-preview is enabled, so swiping back reveals where you're heading before you commit to the gesture.

Performance

⚡

DNS Prewarm

At app startup the currently-selected provider's hostname is resolved on a background thread, so the first rate refresh doesn't have to wait on DNS.

Privacy & Permissions

🔒

Single Permission

Only INTERNET is requested — to fetch exchange rates. No location, no contacts, no camera, no microphone.

🚫

No Tracking

Zero advertising SDKs. Zero analytics. No crash reporter. Your usage never leaves your device.

💾

Local Storage Only

All data (cached rates, preferences, starred currencies) is stored in SharedPreferences on your device. Nothing is synced to any server.
